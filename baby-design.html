Background

A Baby Was Born

My company spun up a design system that covered most of our use cases when it came to components. When multiple designers started using it, we found that it wasn't easy to switch variants because you'd have to go into the layers to toggle the visibility. We ended up slowly rebuilding components to be more straightforward and future-proof.

Fueled by the struggle of working out of that design system, I created my own design library. This personal project became lovingly known as Baby Design.

Documentation of all the available components in the kit.

Baby Design in Darkmode: Blue.

This is still an ongoing project that I revisit from time to time. I'll be keeping an eye out on Figma's updates to see if there's anything I can apply to Baby Design. Check it out on Figma Community!
