Fluppy Cat Mobile Game

Mobile, iOS, Game Design, Visual Design, Illustration

The Challenge

Designing a Whimsical iOS Game

After spending a whole week trying out different types of mobile games, I was inspired and challenged to create my own.

My goal was to create an iOS game that had simple controls and memorable, eccentric visuals.

I was influenced by Flappy Bird and Doodle Jump because they're both famous games yet involved only simple gameplay with little to no instructions. I wanted my own game to be a great representation of mobile user experience.

The Design

Game Plot

The game mechanics are borrowed from Flappy Bird, a side scroller where the player has to tap on the screen repeatedly to keep the sprite from crashing into obstacles. I decided that Pibby (named after my late orange tabby) was going to be the main character and he would fly on a magic blanket, navigating between stalactites and stalagmites to collect fishbone orbs.

As the visual and interaction designer, I had to follow specific iOS guidelines for app design. There would be 3 versions in order for the assets to scale well. The assets included the character sprite, backdrop, app icon, logo and title screen, and other game objects.

Testing the Game

The game was developed using xCode. We even got the sound bit of the chime whenever Pibby collects a fishbone. Once we installed it and gave it a test run, we soon realized that there was a huge lag during the gameplay. After doing some research, it turns out that the the lag was caused by the triangular-shaped obstacles.

Triangles are difficult to generate, therefore taking a longer time to load. In addition to the lag, I realized that it was also difficult to see the sharp points of the the purple spikes, making the game extremely hard.

The solution: get rid of the triangles use rectangles. I came up with the idea to use crystal pillars instead because it would be easier to see against the background and it will remove the lag.

The old version with the stalactites and stalagmites

Blue crystal pillars that are easier to see

The Takeaway

Understanding Game Development

This was my first successful iOS-specific game that I designed. It was a fun side project that allowed me to learn about design constraints and performance issues for iOS, and game development in general. Even till this day, we show the game to friends and strangers who laugh and end up trying it out.
