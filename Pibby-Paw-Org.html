Pibby Paw Org
Visual Design | UX Design | Desktop

Project Overview

As one of the requirements for my UX design certificate, I had to design a website for the use of either ecommerce, booking travels, or pet adoptions. Of course being the cat-lover that I am, I went with the latter.

The required pieces of the assignment include:

  • Homepage
  • Content submission
  • Resources
  • Help section
  • Search feature
  • One internal page of my choice

The entire project took about a month to complete and involved constant communication with my instructor to get feedback at every step of the way.

Website Layout

The design features several common web patterns including:

  • Search feature to find pets on the homepage
  • Autocomplete paired with the search function
  • Dropdowns paired with filters
  • Virtual scrolling for testimonies (carousel with dots)
  • Pagination for search results

Implementing these design patterns would help the website serve its visitors by creating a familiar flow. The main users of this website include prospective adopters, so the search feature would assume a huge role.

I explored every step of the design process and drew a lot from my academic knowledge obtained throughout the course.
